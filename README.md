# 一个项目   
### 为啥要做这个
    啊  我喜欢的抖音视频怎么被下架了！怎么失效了！可恶  手机内存又满了！
### 简介
    一个视频下载汇聚类功能  将视频推送给 应用目前已经支持的下载器 目前仅支持Aria2 并建立视频资源缓存 后台可进行管理查询查看等

### 功能点
    已实现部分
        配置下载器
        多用户管理
        视频资源缓存查看
    未实现部分
        视频墙
        .....
### 技术框架
spring boot 2.7.10
spring boot jpa
sqlite
